**Le Petit Français**

Le Petit Français is an easy-to-use French language-learning software that makes learning basic vocabulary and phrases easy and fun. Le Petit Français does not have a complex interface as all you need is a browser because nothing needs to be installed or set up on a server. The colors of the program are warm red and yellow which give an impression of a friendly atmosphere, reminiscent of French cafes rather than classrooms.

Characteristics
Guided Lessons  
In this mode, ten French words are introduced in each lesson. All words include native-like pronunciation so that learners can practice hearing words correctly. Learners can take a short test after checking new vocabulary. Each lesson completion earns some stars that will be used as the progress marker.

Endless Quiz Mode  
When learners complete one lesson, an endless quiz mode unlocks for them. In this mode, the application asks questions about vocabulary learned in previous lessons in two ways: French to English and English to French. The game continues endlessly giving learners an opportunity to create a streak and consolidate information.

Listening Game  
In this mode, learners develop listening skills as the only source of learning in the application. The application pronounces a French word and a learner should choose its right translation from the list.

Pair Matching Game  
Learners need to pair French words with their English translation.

Survival Phrases Section  
This section provides necessary everyday phrases that are frequently required by beginners right from the start. Each phrase comes with pronunciation tips in order to help users to be more confident while speaking although they are not familiar with French phonetics.

Progress Tracker  
The app features an easy-to-use system of progression, which rewards the user with XP points for each task completed. When learners earn enough XP points, they will advance through themed levels starting from Bébé Baguette up to Légende Française. All the progress is stored in the local browser.

Bilingual Interface  
At any moment, the learner can change the language of the interface from French to English using the toggle button located at the top of the screen.

Running the Application
It is important to note that Le Petit Français was deliberately created with simplicity in mind, and thus launching the program is very straightforward. Installation, command line and server settings are not needed, as all the work has been done inside an HTML file.

Download the project folder or the francais.html file.

Find the downloaded file on your device.

Double-click index.html.

The application should launch in your default web browser automatically.

That’s all there is to it! All processing takes place locally.

It is crucial that you have at least one French voice configured for your device, since the application relies on the speech synthesis capabilities of the web browser. On a Mac, this can be done by going to System Settings > Accessibility > Spoken Content > Voices.

Usage of the Application
Once the application is launched, it is possible to start using it right away.

Select a lesson from the main menu. In each lesson, there are ten words with audio.

Listen to the words and say them out loud.

Pass the mini-quiz to get stars and access other modes.

Having completed at least one lesson, try out the endless quiz, listening game, and matching game.

Visit the survival phrases section anytime to learn how to say useful things in different situations.

Check your XP and level progression in the progress panel.

Change the language of the application interface using the switch in the top menu of the page.

Tips for Learning
Out-loud pronouncing of words is highly recommended. Speaking words significantly improves pronunciation if learners pronounce them, rather than simply read or listen to the words. It is much more effective to have short and regular practice sessions than long and rare ones. Five-minute daily sessions are usually more efficient than hour-long weekly practice sessions.

If audio does not play, check the settings of the speech synthesis and make sure that there are some voices of French language installed. The application uses the speech engine of the browser of itself so it is pretty simple to navigate through.
